# Sentiment Analysis of Real-time Flipkart Product Reviews

## Objective
Classify Flipkart product reviews into Positive and Negative sentiments and identify customer pain points.

## Dataset
Three datasets were provided:
- Badminton reviews
- Tawa reviews
- Tea reviews

All datasets were combined for model training.

## Approach
- Text Cleaning & Lemmatization
- TF-IDF Feature Extraction
- Logistic Regression Classifier
- Evaluation using F1-Score

## Deployment
The application is deployed as a Streamlit web app for real-time sentiment analysis.

Due to AWS account constraints, deployment is done on Streamlit Cloud.
The same application is AWS EC2 compatible without any code changes.
